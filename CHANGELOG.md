# Changelog

All notable changes to PBeep are documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/).

## [1.0] - 2025-01-15

### Added
- Initial public release
- Play system beeps from the command line
- Configurable frequency (Hz or musical notes such as A4, C#3)
- Configurable duration, volume, beep count, and pause between beeps
